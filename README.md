# esp32_ws2812_webservice

Webservice for manage esp32/ws2812 devices

The pictures are created by AI, some other pictures referenced by html are not included. You can create your own.

## index.html

Main page for the web service. It reads a JSON file `files/mainconfig.json`.

It displays several images that you can click on to open the corresponding website in full screen mode.

You can leave this website by pressing escape or click the close button

## dummy.html

A placeholder if the wanted website doesn't exist yet.

## device.jpg

An image for connecting to the main page of a LED device.

It was generated by Gemini AI

## default.png

A placeholder shown if the configured image doesn't exist. It's our cat Simba.

## christmas.html christmas.jpeg

A main page for the christmas season. It shows four images. If you activate an image it plays some music and starts the LED devices.

It's intended to use on server with an VLC media player over his web interface and uses a lighttpd webservice with a special proxy configuration to map the LED device ip adresses on special paths.

I use an Raspberry Pi 3B
